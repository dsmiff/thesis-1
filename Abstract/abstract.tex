\begin{abstract}
Searching for susy at the LHC using the CMS detector. This will cover the all hadronic channel 
search using the \alphat kinematic variable. A full description of the analysis 
method is given as well as results and interpretations within simplified model 
spectra and a the pMSSM.
\end{abstract}
